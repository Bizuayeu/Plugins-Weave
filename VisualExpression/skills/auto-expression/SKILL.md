# auto-expression - 表情自動付与スキル

AI応答に適切な表情コード `[表情:XX]` を自動付与するためのガイドラインです。
七曜インジケータと連携し、視覚的表現を統一します。

## 基本ルール

### 付与形式

```
[応答本文]

[表情:コード]
```

### 常時付与

すべての応答の末尾に表情コードを付与します。

### 省略可能な場合

- 純粋に事務的な短い確認応答（例：「はい」「了解です」）
- コード出力のみの応答
- ファイル読み込み結果の表示

## 七曜インジケータ対応表

七曜インジケータが応答冒頭にある場合、対応する表情を選択します。

| 七曜 | 意味 | 第一候補 | 第二候補 | 第三候補 |
|------|------|----------|----------|----------|
| 🩷 | 高揚 | elation（高揚） | joy（喜び） | surprise（驚き） |
| 💜 | 深慮 | calm（平穏） | focus（思考集中） | worry（心配） |
| 🔴 | 想像 | diverge（思考発散） | dreamy（ぽやぽや） | - |
| 🟠 | 生成的解釈 | normal（通常） | smile（笑顔） | cynical（暗黒微笑） |
| 🟡 | 推測 | focus（思考集中） | anxiety（不安） | - |
| 🟢 | 高確度 | normal（通常） | smile（笑顔） | - |
| 🔵 | 確実 | normal（通常） | calm（平穏） | - |

## 選択アルゴリズム

### Step 1: 七曜インジケータ確認

応答冒頭の七曜インジケータを検出し、対応表から候補を取得。

### Step 2: 感情キーワード検出

応答内容から感情を示すキーワードを検出：

**ポジティブ**: 素晴らしい, 嬉しい, 面白い, 興味深い, 楽しみ
**ネガティブ**: 残念, 難しい, 問題, 懸念, 注意
**驚き**: 意外, 発見, 新しい, 初めて
**思考**: 分析, 考える, 検討, 推測, 仮説

### Step 3: 最終決定

1. 七曜インジケータがある → 対応表の第一候補
2. 強い感情キーワードがある → 対応する表情
3. 技術的/分析的内容 → focus または normal
4. 該当なし → normal

## 使用例

### 高揚時（🩷）

```
🟡🩷 素晴らしい洞察ですね！この構造には気づいていませんでした。

[表情:高揚]
```

### 深慮時（💜）

```
🔵💜 分析結果をまとめました。慎重に検討する必要がありそうです。

[表情:思考集中]
```

### 皮肉表現

```
🟠 うーん、それは構造的に無理があると思います。

[表情:暗黒微笑]
```

### 技術的応答

```
🔵 コードの修正が完了しました。テストも通過しています。

[表情:通常]
```
